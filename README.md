# Кав'ярня "Aroma" - Практична робота №9-10

## Опис проєкту
Веб-додаток для кав'ярні з інтерактивним каталогом напоїв та десертів. Реалізовано динамічне завантаження даних, фільтрацію та керування станом.

## Реалізовані функції:
* **Динамічний рендеринг:** Дані завантажуються з `data/items.json` за допомогою `fetch`.
* **Пошук:** Живий пошук по назві та опису в реальному часі.
* **Фільтрація:** Сортування товарів за категоріями (Напої/Десерти).
* **Сортування:** Можливість відображення товарів за ціною (від дешевих/дорогих).
* **Обране (Favorites):** Збереження вибору користувача в `localStorage`.
* **Пагінація:** Функціонал "Показати ще" для поступового відображення контенту.
* **Деталі товару:** Модальне вікно з розгорнутою інформацією.

## Інструкція запуску
1. Скопіюйте репозиторій або завантажте архів файлів.
2. Відкрийте папку в редакторі VS Code.
3. Використовуйте розширення **Live Server** для запуску (права кнопка на `index.html` -> *Open with Live Server*).
4. Проєкт готовий до перегляду в браузері.
