# Практична робота №2: Основи CSS ☕

**Тема:** Оформлення багатосторінкового сайту кав'ярні 

## Що реалізовано в межах роботи:

### 1. Дизайн та Стилізація (CSS)
- Створено єдину дизайн-систему на основі CSS-змінних (`:root`) для швидкої зміни кольорів.
- Використано блокову модель (Box Model) з налаштуванням `border-box`.
- Оформлено тексти: підібрано сучасні шрифти, налаштовано міжрядкові інтервали та відступи.
- Впроваджено стиль "Карток" (`.card`) з тінями та заокругленням кутів для всього контенту.

### 2. Структура та Навігація
- Горизонтальне меню навігації реалізовано за допомогою Flexbox.
- Додано стани `:hover` для посилань та кнопок, а також клас `.active` для поточної сторінки.

### 3. Медіа та Форми
- Стилізовано відео та аудіо: вони стали адаптивними (`max-width: 100%`).
- Повністю оформлено контактну форму: додано відступи, стилі для полів вводу та ефекти фокусу (`:focus`).

### 4. Адаптивність (Responsive) — ОБОВ'ЯЗКОВО
- Використано Media Queries для екранів менше 768px.
- На мобільних пристроях меню стає вертикальним, а кнопки розтягуються на всю ширину для зручності натискання.

### 5. Бонусні завдання (Виконано)
- Додано плавні переходи (transition) для всіх інтерактивних елементів.
- Використано псевдоелементи `::before` (іконки біля заголовків).
- Застосовано псевдокласи `:nth-child` для стилізації списків та `:hover` ефекти підняття карток.

## Як переглянути:
1. Клонуйте репозиторій.
2. Відкрийте `index.html` у браузері.
