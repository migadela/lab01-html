# ☕ Кав'ярня "Aroma" - Практична робота

## Опис проєкту
Це багатосторінковий веб-сайт для кав'ярні "Aroma", створений у рамках практичної роботи. Проєкт демонструє використання HTML, CSS та інтерактивного JavaScript.

## 🚀 Реалізовані функції (згідно з методичкою):
1. **Динамічна навігація:** Підсвічування активної сторінки в меню залежно від URL.
2. **Мобільне меню:** Кнопка-гамбургер (☰) для відкриття/закриття меню на мобільних пристроях.
3. **Темна тема:** Можливість перемикати світлу/темну тему. Вибір користувача зберігається у `localStorage`.
4. **Кнопка "Вгору":** З'являється при прокручуванні сторінки вниз для швидкого повернення на початок.
5. **Динамічний рік:** Автоматичне оновлення поточного року у футері (підвалі) сайту.
6. **Акордеон:** Розгортання та згортання секції "Наші принципи" на сторінці "Про кав'ярню".
7. **Фільтрація:** Інтерактивне меню на Головній сторінці (фільтр по категоріях "Напої" та "Десерти").
8. **Форма зворотного зв'язку:** - Базова валідація полів.
   - Лічильник символів для повідомлення (максимум 200).
   - Збереження введеного тексту у вигляді "чернетки" (відновлюється після оновлення сторінки).
   - Імітація успішної відправки.
9. **Модальне вікно:** Спливаюче вікно з пропозицією знижки, яке закривається на хрестик, кліком поза вікном або клавішею `Escape`.

## 🛠 Інструкція із запуску
1. Схилюйте репозиторій або завантажте архів з файлами.
2. Відкрийте папку проєкту у редакторі коду (наприклад, VS Code).
3. Для коректної роботи JavaScript (зокрема модулів або локального сховища) рекомендується запускати проєкт через локальний сервер. 
4. Встановіть розширення **Live Server** у VS Code, натисніть праву кнопку миші на файлі `index.html` та оберіть "Open with Live Server".
