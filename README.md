# Практична робота №1: Основи HTML

**Тема:** Структура документа, контент, форми та медіа.
**Тематика сайту:** Кав'ярня "Ароматна Кава"

## Що реалізовано:
- Головна сторінка (`index.html`) зі списками, зображенням та інтерактивною кнопкою (JS).
- Сторінка "Про кав'ярню" (`about.html`) із вбудованими відео та аудіо.
- Сторінка "Контакти" (`contact.html`) із формою зворотного зв'язку та валідацією.
- Використано правильну семантику HTML5 (`<header>`, `<main>`, `<nav>`, `<footer>`).
- Виконано додаткове завдання з JavaScript.

## Як запустити:
1. Завантажте папку з проєктом.
2. Відкрийте файл `index.html` у будь-якому сучасному браузері (Google Chrome, Edge тощо) або використайте розширення Live Server у редакторі коду.
