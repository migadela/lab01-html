Кав'ярня "Aroma" — Керування каталогом (CRUD)
Це фінальна версія веб-додатка для керування меню кав'ярні, розроблена в рамках практичної роботи №11-12. Додаток взаємодіє з REST API для виконання повного циклу операцій з даними (Create, Read, Update, Delete).

Як запустити проєкт
Встановіть Node.js, якщо він ще не встановлений на вашому комп'ютері.

Встановіть json-server (через термінал):

Bash
npm install -g json-server
Запустіть локальний сервер:
Перейдіть у папку з проєктом через термінал і виконайте:

Bash
json-server --watch db.json --port 3000
Відкрийте проєкт:
Відкрийте файл index.html у вашому браузері.

📋 Реалізований функціонал
 GET: Динамічне отримання та відображення списку напоїв та десертів.

 POST: Форма для додавання нових товарів у меню.

 PATCH: Функціонал редагування існуючих записів.

 DELETE: Видалення товарів із попереднім підтвердженням.

 Стани: Реалізовано повідомлення про завантаження, помилки та успішні операції.

 Валідація: Перевірка даних форми перед відправкою на сервер.
